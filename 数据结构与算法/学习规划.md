# Step.2 卡码网

>学习时间：5 天

- 核心代码模式：给你一个函数，直接写函数实现（如：力扣）
- ACM 模式：程序头文件、mian 函数、数据的输入输出都要自己处理

- 比赛、笔试：基本都是 ACM 模式
- 面试：可能是核心代码模式，也可能是 ACM 模式

完成卡码网前 25 道题，就把所有的 ACM 输入输出方式都练习到位了，不会有任何盲区

- [网站](https://kamacoder.com/)
