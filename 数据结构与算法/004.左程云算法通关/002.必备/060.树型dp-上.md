# 树型 dp

本节课讲述最常见的树型 dp 问题，详解树型 dp 的解题套路

下节课会讲述树型 dp 利用 dfn 序的内容

## 树是一种有向无环图

头节点没有父亲，其他节点只有一个父亲的有向无环图，直观理解为发散状

在树上，从头节点出发到任何节点的路径是唯一的，不管二叉树还是多叉树都如此

树型 dp 在树上做动态规划，依赖关系比一般动态规划简单，因为绝大部分多数都是父依赖子

注意：只是依赖关系简单，不代表题目简单

## 树型 dp 套路

1. 分析父树得到答案需要子树的哪些信息
2. 把子树信息的全集定义成递归返回值
3. 通过递归让子树返回全集信息
4. 整合子树的全集信息得到父树的全集信息并返回
