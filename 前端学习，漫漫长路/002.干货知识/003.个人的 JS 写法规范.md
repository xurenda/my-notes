# JS 是否需要分号

知乎：[https://www.zhihu.com/question/20298345](https://www.zhihu.com/question/20298345)

总结：
JS 不需要加分号，除非下一行的第一个 `token` 是 `( `、`[` 、`/` 、`+` 、`-` 这五个字符之一。
```javascript
let a = 0
  , b = 5

; ++a
console.log(a)
```
